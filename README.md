# 🌤️ Dynamic Weather App

> 🔴 **[Live Demo: Click here to view the Weather App in action!](https://devesh-yadav22.github.io/weather-app/)**

A clean, responsive, and dynamic web application that provides real-time weather information for any city in the world. Built using vanilla web technologies and the OpenWeatherMap API.

## ✨ Features
- **Real-Time Data:** Fetches live weather conditions, temperature, highs/lows, and location data.
- **Dynamic UI:** The user interface updates instantly without reloading the page using AJAX (`fetch` API).
- **Clean Design:** Modern, card-based glassmorphism aesthetic with CSS gradients and soft shadows.
- **Error Handling:** Gracefully handles invalid city searches and alerts the user.

## 🚀 Getting Started
To run this project locally, simply clone the repository and open the `index.html` file in your browser. No local server is required. 

*(Note: If you are cloning this repository to edit it, you will need to replace the API key in `script.js` with your own free key from OpenWeatherMap).*

## 💡 What I Learned
During the development of this project, I gained practical hands-on experience with:
