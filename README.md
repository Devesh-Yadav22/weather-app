# 🌤️ Weather App

> 🚀 **[Live Demo: Click here to view the Weather App on Render!](https://weather-app-9wb7.onrender.com)**

A clean, responsive application that provides real-time weather information for any city in the world. Built using vanilla web technologies and the OpenWeatherMap API.

## ✨ Features
- **Real-Time Data:** Fetches live weather conditions, temperature, highs/lows, and location data using the OpenWeatherMap API.
- **Dynamic UI:** The interface updates instantly without reloading the page using the `fetch` API.
- **Modern Aesthetic:** Features a glassmorphism design with CSS gradients, custom fonts, and a responsive layout that works on mobile and desktop.
- **State Management:** The weather display remains hidden until a successful search is performed, ensuring a clean initial user experience.
- **Error Handling:** Built-in validation to alert users if a city is not found or if there is a network error.
